우분투에서는 계정을 생성하는 명령어가 2개

	adduser -> 명령어 실행시 기본 계정정보를 같이 생성
	useradd -> 계정만 생성하며, 기타 다른 정보를 수동으로 생성 및 설정

1. adduser (계정이름)
	-> 계정생성시 비밀번호까지 입력받으며 기본정보를 바로 입력시켜줄수있고, 홈 디렉토리 또한 자동으로 생성
	   계정생성시 우선 암호입력창이 뜨고 후에 부가적인 Commant 입력창이 생성된다.
	   
2. useradd (계정이름)
	-> 계정만 생성해주고 기본 셀인 sh가 할당되며, 홈 디렉토리와 패스워드는 따로 설정

3. usermod (옵션) (계정이름)
	-> 옵션과 계정이름을 받아서, 옵션의 명령에 따라 수정
	
	-c : 계정 Comment를 수정
	-d : 계정 홈 디렉토리를 수정
	-e : 계정 만기 날짜를 수정
	-L : 사용자 계정을 잠근다
	-U : 잠금처리되어 있는 계정을 풀어준다
	-h : 명령어 옵션 설명을 표시
	
4. passwd (계정이름)
	-> 계정 비밀번호를 수정시 usermod -p를 사용하면 /etc/shadow를 직접 바꾸게 되기에 실제로 바꾼 비밀번호로 로그인이 불가능
	   그래서 passwd 명령어를 사용하여 수정해야한다.

5. deluser (계정이름)
	-> 기본적으로 계정만 삭제되며, 계정에 관련된 폴더(파일)을 삭제하려면 옵션을 주어야 한다.
		ex.) deluser (계정이름)
			 deluser -remove-all-files (계정이름)
