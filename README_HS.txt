*read me*

led -> GPIO 21
motor -> GPIO 17
button -> GPIO 24

이 모듈은 버튼을 누르면 모터가 작동하며, 작동하는 시간동안 led에 불이 들어오는 모듈입니다.

사용가능한 사례 : 버튼을 누르면 작동하는 자동문, 
		  댐의 문을 여는 시설, 
		  프로젝트빔을 받는 프레젠테이션의 동작 장치


모터에 레일(rail)이나 벨트(belt)를 연결하여, 일정주기의 동작장치에 응용될수 있다.


추가 가능한 심화 응용 : 모터 작동시 소리 출력 (+sound sensor)
			버튼을 추가하여, 각각의 버튼이 on/off 기능을 하도록 설정 (+button)

심화응용 사용 예시 : 댐의 문을 여는 장치
			-> 1. 수압센서를 추가하여, 수압을 체크 
			   2. 특정 수압 이상의 상태에서 경고음 출력
			   3. 관리자가 경고음을 듣고 버튼 작동
			   4. 경고음 off, led on, 모터 on
			   5. 특정 수압 이하로 내려가면, 모터 원상태로 동작후 off, led off
